**Временная сложность**: $O(n)$

**Пространственная сложность**: $O(n)$

Алгоритм:
- Ищем минимальное значение координаты $x$ среди всех точек ($minX$)
- Ищем максимальное значение координаты $x$ среди всех точек ($maxX$)
- Записываем все исходные точки в мапу
- Ищем координаты оси симметрии как $(minX + maxX)/2$
- Бежим по исходному массиву и для каждой точки ищем в мапе симметричную точку
  - Если в мапе такой точки нет, возвращаем $false$
  - Если пробежались по всем элементам массива, возвращаем $true$
