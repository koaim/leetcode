**Временная сложность**: $O(n)$

**Пространственная сложность**: $O(n)$

Алгоритм:
- Создаём пустую мапу (ключ - элемент массива, значение - индекс элемента в массиве)
- Последовательно обходим массив
- Каждый раз проверяем, есть ли в мапе этот элемент
  - Если такой же элемент есть, сравниваем разницу индекса текущего элемента и индекса элемента из мапы
    - Если эта разница $<=k$, возвращаем `true`
  - Если такого же элемента нет, просто добавляем его в мапу
- В конце возвращаем `false`
